    回到最初的起点

    龙龙

    2017-01-09 16:05

    时间才刚刚过去半年，一切感觉都不对了。没怎么碰代码的自己，在过去的半年里，除了一个字幕生成器以外，几乎没有什么完整的代码。虽然毕业以后，一直在忙碌工作的事情，适应新的环境，还出差学习了视频后期的基础知识，但不写代码的日子，总是在心里缺少些什么。少了刚上大学时“每天一个程序，每个程序都是传奇”的自信，少了争论哪种编程语言最好用时的面红耳赤，少了折腾代码解决bug时的那种驱动感。不过，单纯的我单纯地相信“只要努力，什么时候开始都不晚”，2017年，刚刚忙完教师证，买了新的笔记本，回到最初的起点，重做自己的技术梦。不管以后走不走开发这条路，有一颗热爱技术的心总是好的，因为热爱技术的人往往很可爱~~就像我们几个可爱的小伙伴…………

    Ubuntu14.04关闭蓝牙

    龙龙

    2015-09-21 21:29

    一般很少用蓝牙，开着又费电。

    在Ubuntu14.04中默认关闭蓝牙的方法：

    >> sudo emacs /etc/rc.local
rfkill block bluetooth
exit 0
>> sudo reboot

    KVM克隆虚拟机

    龙龙

    2015-09-16 15:15

    1、启动virt-manager，右击cn001，点击"Clone..."，在弹出窗口的Name中输入"cn002"，点击"Clone"。

    2、克隆完成后，启动cn002。

    3、修改hostname

    >> sudo emacs /etc/hostname
cn002

    4、修改IP地址

    >> sudo emacs /etc/network/interfaces
address 192.168.1.212

    5、重启就可以正常使用cn002了。

    使用pexpect远程登录SSH

    龙龙

    2015-09-16 14:55

    pexpect可以自动和命令行程序进行交互，在系统部署和测试的时候非常好用。

    以远程登录SSH并执行命令为例

    >> emacs dparkstarter.py
import pexpect

ip = "192.168.1.211"
name = "long"
child = pexpect.spawn('ssh  %s@%s' % (name, ip))
child.expect('$')
child.sendline('python dparkslave.py')
child.interact()
>> python dparkstarter.py

    程序可以自动登录SSH，然后执行python dparkslave.py这条命令。
