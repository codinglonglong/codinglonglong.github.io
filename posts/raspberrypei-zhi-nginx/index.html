Raspberry配置nginx

龙龙

2017-02-27 21:51

Source

1、安装nginx。

>> sudo apt install nginx

2、修改主配置文件。

>> sudo emacs /etc/nginx/nginx.conf
user pi;
C-x C-s
C-x C-c

3、修改网站配置文件。

>> sudo emacs /etc/nginx/sites-available/default

4、让网站配置文件生效。

>> sudo ln -s /etc/nginx/sites-available/default /etc/nginx/sites-enabled/default
>> sudo service nginx restart
