Ubuntu命令行连接wifi

龙龙

2017-07-19 11:06

Source

>> cd /usr/local/bin/
>> cd /home/long
>> wpa_passphrase SSID PWD > /home/long/wifi.conf
>> sudo emacs /etc/network/interfaces
auto wlan0
iface wlan0 inet dhcp
wpa-conf /home/long/wifi.conf
>> sudo reboot

注意

路由器必须要打开DHCP才可以正常连接。
