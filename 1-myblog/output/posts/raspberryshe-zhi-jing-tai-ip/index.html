Raspberry设置静态IP

龙龙

2017-02-27 19:45

Source

1、编辑网络配置文件。

>> sudo vi /etc/network/interfaces
iface eth0 inet static
address 192.168.1.254
netmask 255.255.255.0
gateway 192.168.1.1
dns-nameservers 114.114.114.114
>> sudo service networking restart

2、重启系统和路由器。

如果不行的话，试试把路由器也设置成静态地址绑定……
