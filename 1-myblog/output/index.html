    新班主任笔记——入学第二个月（2）

    龙龙

    2017-11-01 23:34

    入学第二个月，没有太多的流程要走了，但是和学生们在一起的生活开始了。学习、卫生和纪律，三个重要的方面要考虑，学习上有时间就多陪陪孩子们，卫生方面很高兴有个负责的卫生委，纪律方面就需要多费心了，目前还在摸索中……

    为了能多陪孩子们学习，很多晚上没有晚自习也留在学校和孩子们一起上课，等到回家，家人都快睡觉了……家人得不到陪伴，家庭关系的问题也需要进一步考虑，合理地控制工作生活时间，是下一步的重点问题。几乎每天的上下班都是在三首歌中度过的——《最好的未来》《你的样子》《你怎么舍得我难过》，这三首歌充分反映了一个新班主任内心的困扰：都是自己的孩子，每个孩子都要好好的，一个也不能少；但是这样真的挺累的，一个班四十多个人，一颗心分成四十多份，制造并消耗爱的过程，也正是增加忧郁和孤单的过程，和小伙伴们在一起总是努力开开心心的，眼睛里看不到大家，忧郁和孤单就迅速来了；当小伙伴们不听话，违规违纪的时候，一方面是愤怒，但更重的分量是难过，感觉自己努力地用正能量去带着大家好好学习，遵规守纪，长了本事以后找个好工作，但还是有不理你这一套的，觉得你是在糊弄他学习，失望和无奈一起涌上心头……往往在上学的时候，不把未来当回事；可等到当回事的时候，你已经没有回头的机会了……恐怕教育学最应该研究的问题之一，就是如何避免这样的悲剧。

    这一个月，出差加上培训就用去了一半，后面还有听课和录课，当班主任的同时，还要带着三个班的二十二节课，“亲子”和“养子”的矛盾就来了，不得不说，开学这两个月，几乎没有多少能安安静静备课的时间，自己班上发生的各类大事小事都要过班主任，不管是违纪，还是生病，或者其他各类事情，除此之外，学校安排的军训、入学手续、学籍建档、体检、出差、培训、听课、录课等等都需要占用时间，再加上孩子们报名参加的主持人大赛之类的学校活动，类似的这些事情都在占用时间；为了培养孩子们的学习习惯，我还安排了每天半小时的读书计划；为了让孩子们知道上课该干什么，不该干什么，剩余时间中的大部分又用在了陪孩子们上课，帮孩子们建立习惯上。所以，对于上课的这三个班来说，网页课还好，是上学时学过的，也是之前一直在用的；但是CAD，在上学的时候根本没开过这门课，我也是“零基础从入门到精通“的……可想而知，没有时间反复研究那些复杂的技巧题，又不忍心一直让孩子们做简单题浪费生命，所以多次被挂在了课堂上……因为我也不是神童，有些技巧题真的够烧脑……于是新的矛盾就来了，为了让这些孩子们能多学点东西，我自己加高了难度；有限的时间又让我无法专心研究这些难题，不得不怀揣着未被完全验证的思路就去上课了，然后就冷场了……一冷场，能帮着你一起解决问题的孩子只是少数，大部分就开始自由自在地扰乱秩序了，因为“老师也不会做了”……也许这样的问题多教几届孩子，经验积累丰富了就好了，但是对于现在的孩子们呢？

    于是，我问我们班的孩子们，你们说是亲生的重要，还是抱养的重要？孩子们不假思索……

    可我也能这样不假思索吗？

    龙龙

    2017-09-25 12:59

    当一个逻辑思维极其连贯的程序员，开始繁杂琐碎的班主任工作时……各种挑战就开始了……

    接新生三天就累病了……这个月一天都没有休息……断断续续的咳嗽一直没好，嘴巴舌头满是溃疡……

    本来想每天都对工作记一下笔记，就像学编程一样……但是操作起来才发现……开学以来，每天从早晨6:30清醒到晚上22:15到家，除了洗澡睡觉，几乎没有自己的时间来反思……新生工作要做，老生一周20多节课要上……开学前因紧张而失眠，开学后都来不及失眠就睡着了……各式表格满天飞，各大部门跑断腿……最无奈的是典型的“送去材料没有人”……真有一种想给学校做一个信息化系统的冲动……

    import os
from mutagenx.easyid3 import EasyID3
from mutagenx.easymp4 import EasyMP4
baseurl = "C:/musics"
flist = os.listdir(baseurl)
for f in flist:
    fname = baseurl + "/" + f
    if fname.endswith(".mp3"):
        try:
            m = EasyID3(fname)
        except Exception as e:
            print(e)
    elif fname.endswith(".mp4") or fname.endswith(".m4a"):
        try:
            m = EasyMP4(fname)
        except Exception as e:
            print(e)
    else:
        print(fname) + " is unable to be converted."
    m["title"] = ""
    m["album"] = ""
    m["artist"] = ""
    m["tracknumber"] = "0"
    m["genre"] = ""
    m["date"] = ""
    m.save()
